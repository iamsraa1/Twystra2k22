BUG BUSTER

Instructions

  1. Participants need to participate individually and the event contains 3 rounds.
  2. Gadgets are not permitted, systems will be provided.
  3. All required software will be installed priorly in teh systems. Internet will not be permitted.
  4. Participants must work with the provided software, Any other software will not be allowed.
  5. Any malpractices will not be tolerated and if found will be disqualified.

Description

  1. First round is code debugging, the participants will be given a code which contains one or more bugs. The participants must identify the bugs and debug the code. They can select any one language from C, C++, Java for this round.
  2. Second round is circuit debugging, the participants will be given a circuit which contains one or more bugs. The participants must identify the bugs and debug the circuit and simulate the circuit to get the required output. They need to simulate the circuit in provided circuit simulator.
  3. In third round, participants can select either coding or circuit designing, the participants will be given a problem statement. They need to code either in C, C++, Java if they opted for coding else they need to simulate their circuit in the provided circuit simulation software and get the required output.
